# Java-essay

#### 介绍
一个Vue搭建的技术博客网站，java后端技术分享，Spring，SpringBoot，SpringCloud等。

#### 软件架构
软件架构说明
Vue，vuepress，vdoing

#### 安装教程

1.  git clone https://github.com/FireHH/Java-essay.git
2.  npm install
3.  npm run dev

#### 使用说明

1.  docs目录下新建目录即可创建自己的文章
2.  文章采用Markdown文档书写
3.  更多教程参考vuepress官网及主题vdoing官网
